# Lottie Animations directory

Drop here Lottie JSON animations from https://lottiefiles.com (free + commercial license).

## Naming convention
- Scene story: `scene_01.json`, `scene_02a.json`, ..., `scene_12.json`
- Slide realistic: `slide_13.json`, `slide_14.json`, ..., `slide_25.json`

## Priorità nel deck
